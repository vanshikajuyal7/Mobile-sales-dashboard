# 📊 Mobile Sales Dashboard

## Overview
This project is an interactive Power BI dashboard developed to analyze mobile sales data and present key business insights through clear and interactive visualizations. The dashboard helps users explore sales performance, identify trends, and make data-driven decisions.

## Dashboard Features
- 📈 Total Sales, Total Quantity, and Total Transactions
- 📅 Monthly Sales Trend Analysis
- 🏙️ City-wise Sales Analysis
- 📱 Mobile Model Performance
- 💳 Payment Method Distribution
- ⭐ Customer Rating Analysis
- 🎛️ Interactive slicers for Month, Brand, Mobile Model, Payment Method, and Day
